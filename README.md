cmake-docset
============

Create a CMake docset to use in @Kapeli 's [Dash](http://kapeli.com/dash) 

Installation
------------
 - checkout this repo 
 - create a subdirectory 'build'
 - into this subdirectory type ```cmake ..```
 - then ```make intall```
 - you should now have a CMake.docset that can be fead to Dash
